1 .C언어란?
   - 인간과 컴퓨터 사이의 의사소통을 위한 프로그래밍 언어
 
  프로그래밍 언어의 종류
   - C언어 , C++언어,  C#언어,  Java언어 …
 
  프로그램(Program)
   - 프로그래밍 언어로 프로그래밍한 작업 결과 (C프로그램, C++프로그램, C#프로그램, Java프로그램 … )

  프로그래머(Programmer)
   - 프로그래밍을 하는 사람 (C프로그래머, C++프로그래머, C#프로그래머, Java 프로그래머 …)
